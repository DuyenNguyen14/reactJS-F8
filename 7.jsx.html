JSX là gì?

Javascript XML (cấu trúc thẻ mở/ đóng). JSX ra đời để hỗ trợ viết XML/ HTML trong Javascript. JSX không phải là HTML., cần có Javascript, Babel để dùng JSX.

Câu hỏi thường gặp:

1. Tại sao khi code thuần dùng map phải join()...?

2. Gặp lỗi khi render từ mảng (prop "key)?

3. Không render được 2 elements cùng lúc?

1. React element types: string, function/ class

  • Footer - Footer component

Props

1. React Elements

  • Sử dụng props giống như với attribute của thẻ HTML

    Ví dụ: object img có props là src và alt

  • 2 props class, for trở thành className, htmlFor

    Ví dụ: => Trong React elements, attr for của tag lable được viết thành htmlFor

  • Phải tuân theo quy ước có sẵn.

2. React Components

  • Sử dụng props giống như đối số của Function
  • Tự do đặt tên cho props (đặt theo camelCase, có thể bao gồm dấu gạch ngang)
Chú ý:
  • Props "key" là prop đặc biệt
  • Props cơ bản là đối số của Component => props có thể là bất cứ dạng dữ liệu nào
  • Sử dụng destructuring
